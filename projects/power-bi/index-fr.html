Impact des Catégories de Produits sur les Indicateurs de Performance des Entreprises – Projet POWER BI

Ce projet universitaire analyse l'impact de différentes catégories de produits (Fournitures de bureau, Meubles, Technologie) sur les indicateurs de performance des entreprises, incluant le chiffre d'affaires, la quantité de commandes et la rentabilité, couvrant les données de 2014 à 2017.

Contexte

Cette analyse explore les modèles d'achat des consommateurs et les stratégies commerciales pour comprendre comment chaque catégorie de produit affecte la performance globale de l'entreprise.

Objectif

Fournir une analyse détaillée des corrélations entre les catégories de produits et les indicateurs de performance des entreprises, identifiant les tendances qui pourraient guider les décisions commerciales futures et la formulation des stratégies.

Méthode

Les données ont été recueillies à partir du "Jeu de Données d'Analyse Commerciale (2014-2017)" qui inclut des données complètes sur les ventes, les clients et les catégories de produits. Diverses techniques analytiques ont été employées :

  • Analyse des tendances de vente à travers différentes catégories de produits.
  • Évaluation des quantités de commandes pour identifier les modèles de demande.
  • Évaluation de la rentabilité et des stratégies de remises à travers les catégories.

Résultats

Les principales conclusions incluent la technologie générant des revenus plus élevés malgré des quantités de vente inférieures, une demande stable pour les fournitures de bureau avec une rentabilité constante, et les meubles montrant une performance variable influencée par les promotions et les conditions économiques.

Les insights stratégiques suggèrent la nécessité de stratégies sur mesure pour chaque catégorie afin de maximiser la rentabilité et la pénétration du marché.

Outils et Technologies Utilisés

  • Microsoft Power BI pour la visualisation des données
  • Python et SQL pour la manipulation et l'analyse des données
