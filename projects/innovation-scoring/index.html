Innovation Scoring Tool

Client: IP firm; MEDEF and CEA partnership.

Person of Contact (POC): Eric Junca (IP Firm POC); Maxime Rousseau (CEA-MEDEF POC); Om Dighe (Technical POC)

Project Length: Between 11 and 22 days

Context

While working on projects that required assessing potential clients' innovativeness, I identified a critical gap in efficiently profiling companies for an IP firm initially and later for broader applications. Traditional methods were cumbersome, involving time-consuming meetings and manual data collection

Objective

My objective was to create the Innovation Scoring Tool, a comprehensive digital solution to streamline the profiling process. This tool aimed to facilitate quicker, more efficient assessments of companies' innovation levels, tailor support from incubators, accelerators, and funding institutions, and provide valuable data for monitoring the effectiveness of support structures in fostering innovation.

Implementation

Led as Project Manager and Initial Developer, from designing the first prototype to managing advanced development stages with added functionalities.

Method

Initiated with conceptual discussions, moved to prototype development, and continuous enhancement based on feedback to refine the tool's precision and utility.

Result

This digital solution is poised to streamline innovation profiling processes, reduce manual effort, and improve data analysis capabilities, marking a significant shift towards more efficient and strategic innovation support and lead generation.

Tools and Technologies Used

  • React: Used for creating a front-end web application, ensuring a user-friendly UI/UX.
  • react-data-export: Library implemented for exporting data to an Excel database, facilitating data analysis and report generation.
  • Redux: Employed for state management within the admin panel, enhancing security and data handling capabilities.
  • React-otp-input: Utilized for implementing two-factor authentication, securing admin panel access.
  • React testing library: Used for writing unit and integration tests to ensure the reliability and functionality of components and modules.
  • HTML and JavaScript: Fundamental technologies used for developing and structuring the web interface.
