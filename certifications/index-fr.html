Certifications

Fondamentaux de Python

Reconnaissant la maîtrise des concepts fondamentaux de Python, cette certification couvre divers aspects essentiels tels que les variables, les structures de données et la syntaxe fondamentale de Python. Obtenu 2.0 crédits CPE/CPD le 2 octobre 2023. ID de certificat : 3065120.

Fonctions, Conditionnalité et Boucles

Cette certification met en évidence la capacité d'implémenter des fonctions Python, d'utiliser une logique conditionnelle et de contrôler le flux avec des boucles pour résoudre des problèmes complexes. Réalisé 2.5 crédits CPE/CPD le 10 octobre 2023. ID de certificat : 3141944.

Stockage, Transformation et Visualisation des Données

Ce certificat démontre la compétence dans la gestion du stockage des données, la réalisation de transformations de données et la création de visualisations pour interpréter efficacement les données. Obtenu 2.5 crédits CPE/CPD le 5 novembre 2023. ID de certificat : 3279347.

Compréhension des Bases de Données SQL

Ce certificat valide les connaissances dans les bases de données SQL, mettant l'accent sur la gestion des données et les requêtes. Complété avec 1.5 crédits CPE/CPD le 13 mai 2024. ID de certificat : 2828387.
